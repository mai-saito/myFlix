myFlix

This app will provide you information about different movies, directors, and genres. You will be able to sign up, update your personal information, and create a list of your favorite movies.

Description Endpoint URL HTTP method Request body Response body
Get a list of all movies /movies GET None JSON object holding data about all the movies, containing the title, genre and director's name property. Example:
{
title : 'The Greatest Showman',
genre : 'Musical',
name : 'Michael Gracey'
},
Get data about a single movie by title /movies/[title] GET None JSON object holding data about a single movie.
Get data about a movie genre /movies/genres/[genre] GET None JSON object holding data about a movie genre.
Get data about a director /movies/directors/[name] GET None JSON object holding data about a director.
Add a new user /users POST JSON object holding data about the user to add. Example:
{
name: "Paulina Gomez",
password: "12345",
email: "paulina@gmail.com",
date of birth: "01/01/2000"
} 		JSON object holding data about the user that was add, including an ID. Example:
{
id: 1
name: "Paulina Gomez",
password: "12345",
email: "paulina@gmail.com",
date of birth: "01/01/2000"
}
Update a user information /users/info PUT JSON object holding data about the user to add. A text message indicating the user information was updated.
Add a movie to the list of favorites /users/favorite POST JSON object holding data about a movie which users like to add to their list of favorites. JSON object holding data about the movie that was added to the list of favorite.
Delete a movie from the list of favorites users/favorite DELETE None A text message indicating the movie that was removed from the list of favorites.
Remove a user /users/[name] DELETE None A text message indicating the user who was removed
